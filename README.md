### 🚀 About Me

I'm a passionate and self-taught **Full-Stack Developer** obsessed with building modern, animated, and scalable web apps using the **MERN stack** and more. I love turning ideas into clean, performant code—and making UIs feel ✨ alive ✨.

- 💻 **Tech Focus**: `MongoDB • Express.js • React.js • Node.js • Next.js • GSAP`

- 📚 **Always Curious**: Exploring TypeScript, Python, Docker & performance optimization.
